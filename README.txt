How to play :
- unzip "MetaConnect4.zip"
- launch "MetaConnect4.exe"

Description :
Each grids follow classic connect 4 rules.
The goal is to win only in 1 grid.
When you play a token in a column, your opponent will play his next token in grid with same position
(example : if a player play a token in second column, his opponent will play his next token in second grid).
If a grid is full, the game end on a egality.)

Controls :
- Left click to put a token in the selected columns
- Escape to return on previous menu
